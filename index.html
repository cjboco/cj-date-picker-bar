CJ Date Navigation Bar Demo

A jQuery plugin to display a horizontal date picker bar to allow quick and easy date selection.

Example

You can select a date and the table data will change based on your selection (Valid dates are May, June July, August, September and October of 2011). Use the theme switcher to see various styles based on jQuery UI theme builder.

Use

To use, simple add the following HTML to your document.

<div class="cj-datenav"></div>

Then initialize the nav bar like so...

$('.cj-datenav').cjDatePickerBar({
   date: '10/31/2011',     // pass a date string to set initial date
   dateMin: '5/2/2010',    // pass a date string to set initial date
   dateMax: '10/31/2011',  // pass a date string to set initial date
   bigInc: 10,             // the big jump amount, defaults to 10
   tinyInc: 5,             // the tiny jump amount, defaults to 5
   monthNames: ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"],
                           // Optional month name overide. Provided for non-english localization.
                           // Defaults to english short names (i.e. what's listed here)
   showInc: false,         // shows the increments numbers, instead of '<<' '<' or '>' '>>'. (defaults false)
   showFuture: true,       // allows the user to navigate to dates greater than the current date. (defaults true)
   callback: function(dateObj) {
      // ...do your callback stuff here.
      // dateObj = a JavaScript date object.
   }
});

CSS

You can then set CSS rules to define the look and position. The following list shows the custom class names that are used. jQuery UI class name styling is also incorporated to allow further control.

.cj-datenav - The main block.
.cj-datenav .nav-months - The month button set
.cj-datenav .nav-years - The year button set.
.cj-datenav .cj-button - The buttons

An example CSS script might look something like this.

/* Style for the date navigation bar */
.cj-datenav { display: block; width: 100%; height: auto; margin-bottom: 15px; overflow: hidden; }

/* nav-months and nav-years are two spans that surround the button groups */
.cj-datenav .nav-months,
.cj-datenav .nav-years { float: left; }
.cj-datenav .nav-months { margin-right: 10px; }

/* buttons use standard jquery ui theme styling, but you can customize using the cj-button class */
.cj-datenav .cj-button { font-size: 10px; margin: 0 1px; }

Options

You can control the behavior by initializing using any of the following option settings.

Option Description Type Default
date To set the start date, you can pass a date string. i.e. "1/1/2011". (will use current date if not passed) string null
dateMin An optional mimimum date selection. i.e. "1/1/2011". (Overides showFuture if provided.) string null
dateMax An optional maximum date selection. i.e. "1/1/2011". (Overides showFuture if provided.) string null
bigInc Determines the big jump amount for the years. number 10
tinyInc Determines the tiny jump amount for the years. number 5
monthNames Default short month names to use for displaying the month buttons array ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"]
showInc If set to TRUE, then the year increment buttons will show the actual jump value. If FALSE, then uses <<, <, >, >> boolean false
showFuture If set to TRUE, then future dates are navigable. FALSE, prevents the selection of future dates. i.e. Dates greater than the current date boolean true
callback An optional callback function that gets called after a button is clicked. function null
